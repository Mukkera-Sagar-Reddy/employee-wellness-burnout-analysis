# Employee Wellness & Burnout Analysis

## Project Overview
This project analyzes employee wellness, workload pressure, burnout risk, and job satisfaction using Python and Power BI. The objective was to identify the major factors contributing to employee burnout and provide data-driven workforce wellness insights for organizational decision-making.

## Business Problem
TechCorp International observed declining employee engagement despite competitive compensation structures. The organization suspected that external stressors such as workload, sleep quality, and commute pressure were significantly impacting employee satisfaction and burnout levels.

This project focuses on identifying:
- Burnout thresholds
- High-risk departments
- Satisfaction decline patterns
- Wellness-related workforce trends

- Employee wellness analysis

- Department-level burnout comparison
- Sleep vs satisfaction analysis

- Interactive Power BI dashboard
- Business-focused insights and recommendations

## Key Insights
- Higher workload levels significantly reduce employee satisfaction.
- Healthy sleep patterns are strongly associated with higher job satisfaction.
- Marketing and Legal departments show the highest burnout pressure.
- IT, Customer Service, and Marketing remain below the organizational satisfaction benchmark.
